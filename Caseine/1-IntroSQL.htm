    Your progress

    • Qu'est ce qu'est SQL ?

      Le SQL (Structured Query Language) est un langage permettant de communiquer avec une base de données. Ce langage informatique est notamment très utilisé par les développeurs web pour communiquer avec les données d’un site web.

      MySQL, PostgreSQL, SQLite, Microsoft SQL Server ou encore Oracle sont des Système de gestion de base de données (SGBD). Tous comprennent le SQL mais ont certaines spécificités. Ce cours ne rentre pas dans les spécificités des SGBD.


    • C'est quoi une base de données relationnelles ?

      Avant de commencer à apprendre la syntax SQL, il est important de comprendre le modèle mental des base de données relationnelles.

      Une base de données relationnelles peut être représentée par un ensemble d'onglets dans un fichier excel. Chaque onglet représente une table avec un nombre de colonnes fixes qui ont un nom.

      Par exemple, si une entreprise a une base de données de ses clients avec à chaque fois leur prénom, nom, numéro de téléphone, ville et pays alors la table de la base pourrait ressembler à :

    • Comment fonctionne ce cours ?

      Le cours est découpé en étape vous permettant de cibler une ou deux notions propres au langage SQL. Pour chacune de ces notions vous aurez un quizz avec une ou plusieurs questions auto-corrigées vous permettant de valider les notions acquises.
